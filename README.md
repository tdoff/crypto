# Toko application

The back-end application for the Crypto Recommendation project

## Installation
 
Set environment up to run application by starting containers: 

```bash
docker-compose up -d # to download and install images in detached mode
docker-compose ps # to check containers status
```

## Documentation

REST API documentations are automatically generated by Swagger UI plugin 
and available on the root of application. It's `localhost:8080` for locally deployed app
